VanArsdel Sales & Market Performance Dashboard | Power BI

Designed and built an end-to-end interactive Power BI dashboard to analyse sales performance, market share, and revenue growth trends for a fictional FMCG company (VanArsdel Ltd.) across time, geography, product categories, and manufacturers.

Key work:
→ Data cleaning and transformation: handled missing values, standardised formats, and ensured consistency across multiple datasets
→ Built a relational data model connecting Sales, Product, Manufacturer, Geography, and Date tables
→ Developed DAX measures including Revenue, % Growth, and PY (Previous Year) Sales for YoY performance tracking
→ Implemented dynamic filters and drill-down capabilities for multi-level analysis
→ Dashboard enables market share distribution analysis, high-performing region identification, and trend tracking to support data-driven decisions

Tools: Power BI Desktop, DAX, Power Query
